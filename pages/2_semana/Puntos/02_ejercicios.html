Ejercicios XML

1. Catálogo de software informático

Objetivo:

Crear un documento XML con espacios de nombres diferenciados.

Tarea:

  1. Crear un documento XML que contenga información de software de sistemas operativos y de oficina.
  2. Utilizar espacios de nombres para diferenciar los tipos de software.
<catalogo xmlns:so="http://sistemas-operativos.com" xmlns:of="http://oficina.com">
  <so:software>
    <so:nombre>Windows 11</so:nombre>
    <so:creador>Microsoft</so:creador>
    <so:precio>139.99</so:precio>
  </so:software>
  <of:software>
    <of:nombre>Microsoft Office 365</of:nombre>
    <of:creador>Microsoft</of:creador>
    <of:precio>99.99</of:precio>
  </of:software>
</catalogo>

Observa cómo se utilizan los prefijos para diferenciar los elementos de cada espacio de nombres.

2. Información de un libro y su autor

Objetivo:

Separar información de libros y autores usando espacios de nombres.

Tarea:

  1. Crear un documento XML que contenga información de un libro y su autor.
  2. Utilizar espacios de nombres diferentes para la información del libro y del autor.
<biblioteca xmlns:lib="http://libros.com" xmlns:aut="http://autores.com">
  <lib:libro>
    <lib:titulo>Cien Años de Soledad</lib:titulo>
    <lib:anioPublicacion>1967</lib:anioPublicacion>
  </lib:libro>
  <aut:autor>
    <aut:nombre>Gabriel García Márquez</aut:nombre>
    <aut:nacionalidad>Colombiana</aut:nacionalidad>
  </aut:autor>
</biblioteca>

Cada espacio de nombres tiene su propio prefijo para evitar conflictos entre elementos.

3. Documento XML con espacios de nombres diferenciados

Objetivo:

Diferenciar elementos similares con espacios de nombres.

Tarea:

  1. Crear un documento XML que contenga elementos con el mismo nombre pero diferentes significados.
  2. Utilizar espacios de nombres para diferenciarlos claramente.
<documento xmlns:curso="http://curso.com" xmlns:libro="http://libro.com">
  <curso:titulo>Programación Avanzada</curso:titulo>
  <libro:titulo>Gestión Empresarial</libro:titulo>
</documento>

Aunque ambos elementos se llaman "título", los espacios de nombres los diferencian claramente.

4. Análisis de documento XML

Objetivo:

Analizar un documento XML con espacios de nombres.

Tarea:

  1. Analizar el siguiente documento XML e identificar los espacios de nombres y elementos asociados.
<empresa xmlns:emp="http://empresa.com" xmlns:pro="http://proyecto.com">
  <emp:empleado>
    <emp:nombre>Ana López</emp:nombre>
  </emp:empleado>
  <pro:proyecto>
    <pro:nombre>Sistema de Gestión</pro:nombre>
  </pro:proyecto>
</empresa>

Espacios de nombres declarados:

  • http://empresa.com
  • http://proyecto.com
Elementos por espacio de nombres:
  • http://empresa.com: emp:empleado, emp:nombre
  • http://proyecto.com: pro:proyecto, pro:nombre

5. Corrección de errores en el uso de espacios de nombres

Objetivo:

Identificar y corregir errores en el uso de espacios de nombres.

Tarea:

  1. Identificar los errores en el siguiente documento XML.
  2. Corregir los errores relacionados con el uso de espacios de nombres.
<catalogo xmlns="http://biblioteca.com" xmlns:lib="http://tienda.com">
  <lib:titulo>Don Quijote</lib:titulo>
  <autor>Miguel de Cervantes</autor>
</catalogo>

Este documento XML ya está correctamente formado con espacios de nombres adecuados.
